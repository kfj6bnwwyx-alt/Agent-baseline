# Chain: New Component

Trigger: "new component", "build from scratch", "create [X] component"

## Steps
1. metadata-extraction (if Figma source exists)
2. ux-design (interaction spec)
3. ui-design (visual spec, token mapping)
4. code-gen (implementation against contract)
5. accessibility-audit (audit implementation)
6. testing (test plan + tests)

## Context bridges between steps
Each step produces ~500 token summary. Do not carry full conversation.
