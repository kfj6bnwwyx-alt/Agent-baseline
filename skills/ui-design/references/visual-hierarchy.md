# Visual Hierarchy

Adapted from designer-skills (MC Dean): visual-hierarchy.

## Hierarchy tools (in order of strength)
1. **Size**: Larger elements draw attention first
2. **Weight**: Bold text commands attention over regular
3. **Color**: High contrast and saturated colors advance; muted recedes
4. **Spacing**: Isolated elements feel more important
5. **Position**: Top-left (LTR) gets read first

## Typography hierarchy
- **Display/Hero**: One per page. The thing you see first.
- **Heading 1**: Section titles. ~24-32px.
- **Heading 2**: Subsection titles. ~18-24px.
- **Body**: Default reading text. 15-16px. Line height 1.5-1.7.
- **Caption/Meta**: Secondary info. 12-13px. Muted color.

Rule: No more than 3-4 font sizes on a screen. Use weight and
color to create sub-levels within sizes.

## Color hierarchy
- **Primary action**: Most saturated, highest contrast
- **Secondary action**: Lower contrast, often outlined
- **Content**: Near-black on near-white (or inverse in dark mode)
- **Meta/secondary**: Gray or muted tone
- **Danger/warning**: Semantic colors reserved for these meanings

## Whitespace as hierarchy
- More whitespace around an element = more importance
- Group related items with tight spacing
- Separate unrelated groups with generous spacing
- The ratio matters more than absolute values
