# WCAG Checklist (Summary)

Quick checks for non-specialist skills. For full audits, load wcag-checklist.md.

1. **Color contrast**: 4.5:1 for normal text, 3:1 for large text (AA)
2. **Keyboard access**: All interactive elements reachable via Tab, operable via Enter/Space
3. **Focus visible**: Clear focus indicator on all interactive elements
4. **Alt text**: All informative images have descriptive alt text
5. **Labels**: All form inputs have visible, associated labels
6. **Heading hierarchy**: h1 → h2 → h3, no skipped levels
7. **Touch targets**: Minimum 44×44px for mobile
8. **Motion**: Respect prefers-reduced-motion
