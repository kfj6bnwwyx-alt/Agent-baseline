# Cognitive Load

Adapted from inclusive-design-skills (MC Dean).

## Cognitive load types
- **Intrinsic**: Complexity of the task itself (can't reduce, but can scaffold)
- **Extraneous**: Complexity from poor design (reduce this)
- **Germane**: Effort spent building understanding (support this)

## Reduce extraneous load
- Use plain language (reading level 8th grade or below)
- Chunk information (7±2 items per group)
- Provide defaults (don't force unnecessary choices)
- Show progress in multi-step processes
- Keep navigation consistent across pages
- Use familiar patterns (don't reinvent standard interactions)
- Minimize required memory (show info, don't make users remember it)

## Support diverse cognitive styles
- Provide both visual and text explanations
- Allow customizable pacing (no auto-advancing carousels)
- Offer undo for all actions (reduce anxiety about mistakes)
- Avoid time limits where possible
- When time limits are necessary, allow extension
